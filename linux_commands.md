# Linux Commands

## Navigation Commands

### pwd

Purpose:
Shows current directory.

Example:

pwd

### ls

Purpose:
Lists files and folders.

Example:

ls

### cd

Purpose:
Changes directory.

Example:

cd Documents

---

## Directory Commands

### mkdir

Purpose:
Creates a directory.

Example:

mkdir Cybersecurity

### rmdir

Purpose:
Removes an empty directory.

Example:

rmdir Test

---

## File Commands

### cp

Purpose:
Copies files.

Example:

cp file1.txt backup.txt

### mv

Purpose:
Moves or renames files.

Example:

mv old.txt new.txt

### rm

Purpose:
Deletes files.

Example:

rm file.txt

---

## Permission Commands

### chmod

Purpose:
Changes permissions.

Example:

chmod 755 script.sh

### chown

Purpose:
Changes ownership.

Example:

sudo chown root file.txt

---

## Search Commands

### grep

Purpose:
Searches for text.

Example:

grep hello file.txt

---

## Networking Commands

### ping

Purpose:
Tests connectivity.

Example:

ping google.com

### ip a

Purpose:
Shows IP addresses.

Example:

ip a

---

## Package Management

### sudo apt update

Purpose:
Updates package list.

Example:

sudo apt update

### sudo apt install

Purpose:
Installs software.

Example:

sudo apt install nmap
